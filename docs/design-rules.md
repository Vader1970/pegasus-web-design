# Design Rules

## Goal
Build a modern, premium, high-converting website for Pegasus Web Design.

The design should feel:
- clean
- structured
- confident
- professional
- built for real businesses (not designers)

Avoid:
- over-designed UI
- excessive gradients
- visual clutter
- generic template feel

---

## Colour System

### Base backgrounds
- Primary background: #FFFFFF
- Soft section background: #F8F9FB
- Alternate section background: #F1F3F7
- Dark sections: #07132D
- Footer / deep background: #06112A

### Brand colours
- Primary brand: #4F52C9
- Accent purple: #6370FF
- Soft tint background: #F4F5FD

### Text colours
- Heading: #111111
- Body: #3D3D3D
- Secondary: #6B7280
- On dark: #FFFFFF
- On dark secondary: rgba(255,255,255,0.72)

### Borders
- Light border: #E5E7EB
- Medium border: #DDE2EA

---

## Typography

### Font Family
Use:
'Inter Tight', Inter, Arial, sans-serif

---

## Type Scale

### Hero Heading (H1)
- clamp(56px, 6vw, 84px)
- line-height: 0.95 – 1.0
- font-weight: 700–800
- letter-spacing: -0.03em

### Section Headings (H2)
- clamp(36px, 4vw, 56px)
- line-height: 1.0 – 1.08
- font-weight: 700
- letter-spacing: -0.03em

### Card Titles (H3)
- 24px – 32px
- line-height: 1.15
- font-weight: 600–700

### Body Large
- 18px
- line-height: 150%

### Body Default
- 16px
- line-height: 150%

### Small Text
- 14px
- line-height: 150%

### Eyebrow / Label
- 12px – 13px
- uppercase
- letter-spacing: 0.12em
- font-weight: 600

---

## Layout System

### Max Widths
- Content max width: 1312px
- Full section width: 1440px

### Section Padding
- Desktop: 120px 0
- Tablet: 96px 0
- Mobile: 72px 0

### Container Padding
- Mobile: 24px
- Tablet: 32px
- Desktop: 64px

---

## Spacing System

Use consistent spacing scale:

8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96, 120

Never use random spacing values.

---

## Borders & Radius

- Small: 12px
- Medium: 16px
- Large: 20px

Do not over-round elements.

---

## Shadows

### Default
0 8px 24px rgba(0,0,0,0.04)

### Hover
0 12px 32px rgba(0,0,0,0.08)

Keep shadows subtle.

---

## Buttons

### Primary
- Background: #4F52C9
- Text: #FFFFFF
- Padding: 14px 22px
- Radius: 12px
- Font-weight: 600

### Secondary
- Transparent background
- Border: 1px solid #E5E7EB
- Text: #111111

---

## Section Flow

Sections should alternate subtly:

- white
- soft grey
- white
- soft grey
- white
- dark CTA
- light section
- footer

Avoid long runs of identical backgrounds.

---

## Hero Design Rules

Hero must feel:
- spacious
- clear
- premium
- focused

Use:
- strong left text
- right-side visual
- subtle background depth

Do not:
- use logo as background watermark
- clutter the hero
- overuse gradients

---

## Cards

Cards should feel:
- clean
- structured
- lightly elevated

Use:
- subtle borders
- soft shadows
- clear spacing

---

## Forms

Forms should feel:
- simple
- trustworthy
- easy

Use:
- clean inputs
- soft borders
- clear labels
- generous spacing

---

## Final Standard

The design should feel like:
- a premium agency site
- built for real businesses
- focused on results, not decoration
